SamWizard checks your server first, then will walk you through setting up a private Windows file shared drive you can access whenever you'd like.

What you will choose

Which drive you want to access remotely, What you want to NAME that drive, and a private username and password.

What this wizard will do

Install the App SAMBA, mount the share drive with the name you chose, and show the final Windows connection path you use to access your drive.

What is SAMBA?

Samba is an application that manages access to folders and drives from outside the server. You can use BRAND NEW external hard drives, or an existing enternal drive you've already got data on

How and where can I access my drive?

SamWizard will give you the address of where to access your drive on your network. If you want to access your drive from outside your home network you'll need to set up a private encrypted tunnel (So cool, right !?) with a service called TAILSCALE, which you can set up after.
